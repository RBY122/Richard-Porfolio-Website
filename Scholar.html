📘 Lamesa

Scholar is a scalable, modular school management system designed to enhance administrative efficiency and promote effective communication among educators, students, and guardians. It features role-based dashboards, structured data handling, and responsive design for a seamless experience across devices.

🔧 Technologies Used

  • ✔️ ASP.NET Core MVC
  • ✔️ Firebase Authentication & Firestore
  • ✔️ Enums and modular model structure
  • ✔️ Advanced CSS & animation libraries

🎯 Highlights

With intuitive navigation, dynamic user roles, and secure access control, Lamesa provides an engaging and reliable platform tailored to school needs. It’s designed to scale across institutions and adapt to evolving educational demands.
